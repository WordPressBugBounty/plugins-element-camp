ElementCamp is an Elementor addon plugin that provides custom widgets to enhance your website-building experience.

== Description ==

ElementCamp is a lightweight and powerful plugin that extends Elementor's capabilities by adding a custom widget. It's perfect for enhancing the design and functionality of your WordPress website.

**Features:**
- Fully compatible with Elementor page builder.
- Adds a custom widget for dynamic and creative content display.
- Easy to use and customize.

**How to use:**
1. Install and activate the plugin.
2. Ensure Elementor is installed and active.
3. Go to the Elementor editor and look for the "ElementCamp Widget" under the "General" category.
4. Drag, drop, and customize!

== Installation ==

1. Upload the `elementcamp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Ensure Elementor is installed and active.
4. Start building with the new widget in Elementor!
